Fundamentals - DRY: Level 2

You've formatted the title for a warehouse report. Our next use case is to format the titles of SKU and SKU-warehouse reports. The good news (and bad news!) is that code to format it has already been written. However, it contains duplication and your task is to refactor the code to remove as much duplication as you can – without breaking any of the tests.

Pre-existing code

class FormattedTextGenerator:
    TITLE_TAG_FORMAT = "<{0}h3>"
    TITLE_OPEN_TAG = TITLE_TAG_FORMAT.format("")
    TITLE_CLOSE_TAG = TITLE_TAG_FORMAT.format("/")
    
    def formatWarehouseReportTitle(self, warehouseName):
        return self.TITLE_OPEN_TAG + "Warehouse: " + warehouseName + \
            self.TITLE_CLOSE_TAG
    
    def formatSkuReportTitle(self, skuName):
        return self.TITLE_OPEN_TAG + "SKU: " + skuName + self.TITLE_CLOSE_TAG
    
    def formatSkuWarehouseReportTitle(self, skuName, warehouseName):
        return self.TITLE_OPEN_TAG + "SKU: " + skuName + ", Warehouse: " + \
            warehouseName + self.TITLE_CLOSE_TAG
