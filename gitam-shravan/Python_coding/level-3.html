Fundamentals - DRY: Level 3

Now that we've finished with the formatting of report titles, we only have a few minor things to finish up on the formatting side of the inventory reporting system. After that, we can move on to the juicier task of actually creating reports. Use cases:

  • There may be cases where there is an error generating the report. We want to return a special title in those cases: <h3 class="text-error">Error generating your report!</h3>
  • We want to format individual lines of the report. Each line consists of a description and value. If the description is Quantity and value is 10, the formatted line would be <strong>Quantity:</strong> 10
  • Since we're using the : in report lines, we want to change the titles to use a - instead of a :. So, <h3>Warehouse: Agra</h3> becomes <h3>Warehouse - Agra</h3>. The other 2 types of titles will also change similarly.
