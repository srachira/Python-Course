Fundamentals - DRY: GitHub Analysis

String duplication commonly occurs when your code is interacting with information that is coming from a different system. Parsing configuration from a file, reading parameters from HTML forms and sending out status/notification messages are some common use cases that deal with string literals.

We'll now look at some real code examples and try to find out if there's any string duplication going on. Your task is to analyze 2 GitHub files and figure out which string literals show duplication and can be abstracted out. Ignore string literals that are part of error handling/logging.
